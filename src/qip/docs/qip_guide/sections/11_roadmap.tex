\section{Roadmap}

Planned extensions (kept intentionally modular):
\begin{itemize}
  \item richer device mappings from PSD to relaxation/dephasing rates
  \item scheduling-aware compilation and transpilation
  \item optional sparse linear algebra and larger Hilbert spaces
  \item improved noisy gate models linked to Lindblad generators
  \item documentation examples that map directly to case study notebooks
\end{itemize}
